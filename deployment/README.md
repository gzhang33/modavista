## 🎯 项目概述

DreaModa是一个专业的B2B时尚产品展示系统，包含两个核心项目：

### 主项目特性
- **技术栈**: PHP 8.3 + MySQL 8.0 + Apache
- **功能**: 产品展示、多语言支持、管理后台、SEO优化
- **适用**: 传统企业客户、快速上线需求

### SampleShowcase特性  
- **技术栈**: React 18 + TypeScript + Express + Node.js
- **功能**: 现代化UI、实时交互、组件化架构
- **适用**: 技术导向客户、高端用户体验需求

## 🛠️ 常见问题

### Q1: 阶段1和阶段2可以无缝切换吗？
**答**: 可以，但需要注意：
- 阶段1→阶段2：数据可以完整迁移，需要重新配置服务器
- 阶段2→阶段1：主要功能可以保留，React应用功能会丢失
- 建议：先从阶段1开始，需要时升级到阶段2

### Q2: 数据库结构兼容吗？
**答**: 完全兼容
- 两个阶段使用相同的MySQL数据库
- SampleShowcase通过视图和适配层读取DreaModa数据
- 数据修改通过DreaModa管理面板统一管理

### Q3: 域名配置有什么要求？
**答**: 
- 阶段1：标准域名解析到共享主机IP
- 阶段2：域名解析到VPS IP，支持子路径访问
- 两个阶段都支持SSL证书和www重定向

### Q4: 如何处理大量产品图片？
**答**: 
- 阶段1：优化图片大小，使用Hostinger CDN
- 阶段2：可以配置专门的图片CDN服务
- 建议：图片压缩到500KB以下，使用WebP格式

### Q5: 多语言内容如何管理？
**答**: 
- 数据库支持完整的i18n结构
- 通过管理面板可以添加多语言内容
- 系统自动根据用户语言显示对应内容

### Q6: 性能优化建议？
**答**: 
- **阶段1**: 启用压缩、优化图片、使用CDN
- **阶段2**: 同上 + Redis缓存、负载均衡
- **通用**: 定期清理数据库、监控查询性能

### 紧急问题处理

#### 网站无法访问
1. 检查域名DNS解析
2. 确认Hostinger服务状态
3. 查看Apache错误日志
4. 联系Hostinger技术支持

#### 数据库连接错误
1. 检查数据库用户权限
2. 确认连接参数正确
3. 查看MySQL错误日志
4. 重启数据库服务

#### React应用异常（仅阶段2）
1. 检查PM2进程状态：`pm2 status`
2. 查看应用日志：`pm2 logs`
3. 重启应用：`pm2 restart sampleshowcase`
4. 检查Node.js版本兼容性
